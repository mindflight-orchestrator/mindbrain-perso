Graph stream in the browser

Incremental SSE consumer for the `graph.streamSubgraph(...)` contract. Seed nodes arrive first, then nodes and edges, and the UI redraws as each event lands.

idle
The page listens for SSE events named `seed_node`, `node`, `edge`, and `done`. If the stream endpoint is on another origin, the server must allow CORS.
